## 📌 Executive Summary & Platform Vision

**Andhra ISP Network** is an enterprise-grade digital connectivity and Internet Service Provider (ISP) discovery platform engineered to map, index, and optimize broadband infrastructure across all **26 districts, 679 mandals, and thousands of villages in Andhra Pradesh** — designed to scale into India's premier nationwide Digital Public Infrastructure (DPI) layer.

By bridging citizens, local internet service providers (LCOs/MSOs), state infrastructure authorities (AP FiberNet), and enterprise telecom operators, the platform delivers real-time ISP discovery, interactive spatial GIS vector mapping, automated AI verification workflows, and zero-trust security.

| Backend API       | NestJS Microservices, RESTful API Gateway, Redis Caching      |
| Database & Search | PostgreSQL 16 + PostGIS, Trigram Fuzzy Search, pgvector Embeds |
| AI & Automation   | 9-Stage Execution Engine, Conversational RAG Copilot, MLOps   |
| Infrastructure    | Managed Multi-AZ Kubernetes (EKS/GKE), HPA, DevSecOps Scanners|

## ⚡ Pin-to-Pin Quickstart & Installation Guide

Follow these exact steps to clone, configure, and launch the platform locally:

### 1. Prerequisites
Ensure your local development environment has the following installed:
* **Node.js:** `v18.20.0` or higher (`node -v`)
* **NPM:** `v10.0.0` or higher (`npm -v`)
* **Git:** `v2.40.0` or higher (`git --version`)

### 2. Clone the Repository
```bash
git clone https://github.com/Masthan008/Andhra-ISP-Network-.git
cd "Andhra ISP Network"
```

### 3. Install Monorepo Dependencies
```bash
# Install root and workspace dependencies
npm install
```

### 4. Configure Environment Variables
Create a `.env` file in `apps/web/`:
```bash
# Windows PowerShell
Copy-Item apps/web/.env.example apps/web/.env

# Linux / MacOS
cp apps/web/.env.example apps/web/.env
```

Set the required environment keys in `apps/web/.env`:
```env
VITE_APP_TITLE="Andhra ISP Network"
VITE_API_BASE_URL="http://localhost:3000/api/v1"
VITE_ENABLE_ANALYTICS="false"
```

### 5. Launch the Local Development Server
```bash
# Runs the Vite dev server for apps/web
npm run dev
```

Open your browser and navigate to:
👉 **`http://localhost:5173/`**

## 🔒 Security, Compliance & DPDP Standards

The platform strictly adheres to Indian **Digital Personal Data Protection (DPDP) Act 2023** and enterprise DevSecOps standards:
* **Data Minimization:** Zero PII exposure on public endpoints.
* **Encryption:** AES-256 for data at rest; TLS 1.3 for data in transit.
* **Authentication:** OAuth2 Client Credentials for partners; JWT with sliding refresh tokens for users.
* **Security Scanning:** Automated Trufflehog secret detection, Snyk dependency scanning, SonarQube SAST, and Trivy container scanning.
